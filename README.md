# To-Do List CLI
A simple command-line to-do list application with persistent storage.

## Project status
In development - started 23 October 2025

## Planned features
1. View all tasks
2. Add tasks
3. Delete tasks
4. Mark tasks as complete
5. Save / load from file

- JSON for data storage
